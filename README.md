# stroke-dataset
 
Data Source: https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

## How to run:
- R Version 4.2.2+
- Shiny Version 1.7.3+
- Tidyverse Version 1.3.2+
- ggplot2 Version 3.4.0+
- Run the following commands:

```
library('Shiny')
shiny::runGitHub("ATP_WTA_Stats","lunching")
```
